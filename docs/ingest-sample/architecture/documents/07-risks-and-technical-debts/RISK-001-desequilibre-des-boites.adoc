== Risque

Le regroupement par rangs adjacents crée des paliers décroissants : les boîtes n'ont
pas une valeur équivalente. Un participant pourrait percevoir le pool comme
déséquilibré.

== Impact

Perception d'iniquité; choix « évidents » dans les boîtes de tête réduisant l'intérêt
stratégique.

== Atténuation

Assumé pour le POC (voir BR-1.3). Un rééquilibrage en serpentin est identifié comme
évolution possible via la configuration des boîtes (FA2), hors portée actuelle.
